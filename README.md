# Customer Transaction Analysis

## 📊 Overview
This project analyzes customer banking transaction data to identify spending patterns, high-value customers, and potentially suspicious activity.

## 🧰 Tools Used
- Python
- Pandas
- NumPy
- Jupyter Notebook

## ❓ Business Questions
- Which customers have the highest total transaction value?
- What transaction categories move the most money?
- Are there any suspicious or unusually large transactions?
- How do transactions change over time?

## 🔍 Key Insights
- Customer 104 had the highest total transaction amount.
- Salary transactions made up the largest portion of funds.
- A transaction over $1000 was flagged as suspicious.
- Different customers show different transaction behaviors.

## 📁 Project Files
- `transaction_analysis.ipynb` → main analysis notebook  
- `transactions.csv.csv` → dataset used for analysis
